# @ariadnejs/types

TypeScript type definitions for Ariadne.

## Installation

```bash
npm install @ariadnejs/types
```

## Usage

```typescript
import { Def, Ref, CallGraph } from '@ariadnejs/types';

// Use the types in your TypeScript code
const definition: Def = {
  id: 1,
  kind: 'definition',
  name: 'myFunction',
  // ...
};
```
