# @ariadne/types

TypeScript type definitions for Ariadne.

## Installation

```bash
npm install @ariadne/types
```

## Usage

```typescript
import { Def, Ref, CallGraph } from '@ariadne/types';

// Use the types in your TypeScript code
const definition: Def = {
  id: 1,
  kind: 'definition',
  name: 'myFunction',
  // ...
};
```
